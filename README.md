# 🌍 Real-Time Groundwater Level Prediction Using DWLR Data

A machine learning-based web application that predicts groundwater levels using Digital Water Level Recorder (DWLR) data. The system provides predictive analytics, interactive visualizations, and a responsive interface to support groundwater monitoring and sustainable water resource management.

## 🚀 Features
- Groundwater level prediction using Machine Learning
- Interactive Power BI Dashboard
- Historical DWLR data analysis
- Responsive web interface
- Data visualization and trend analysis

## 🛠️ Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
- PHP
- MySQL
- Python
- Power BI

## ▶️ Run the Project
1. Clone the repository:
   ```bash
   git clone https://github.com/rutuja-kadam27/groundwater_prediction.git
   ```
2. Import the database into MySQL.
3. Start Apache and MySQL using XAMPP.
4. Open the project in your browser.
